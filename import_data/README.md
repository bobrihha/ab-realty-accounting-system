# Инструкция по импорту данных

Я проанализировал вашу Google Таблицу. Пожалуйста, экспортируйте следующие вкладки как файлы **CSV** и загрузите их в папку `import_data`.

## 1. employees.csv (Сотрудники)
**Источник:** Вкладка `ПО-ЛИ 2025 (АГЕНТЫ)` или `СДЕЛКИАГЕНТЫ2025` (где есть полный список агентов).
Обязательные колонки (скрипт поймет русские заголовки):
- `Имя` (Полное имя)
- `Email` (Если есть, иначе скрипт может пропустить или создать фиктивный)
- `Телефон`
- `Роль` (Например: Агент)
- `Статус` (Активен)
- `Ставка` (Процент комиссии)

## 2. deals.csv (Сделки)
**Источник:** Вкладка `СДЕЛКИАГЕНТЫ2025`
Обязательные колонки (скрипт поймет эти заголовки):
- `ПОКУПАТЕЛЬ`
- `ПРОДАВЕЦ` или `ОБЪЕКТ`
- `СТОИМОСТЬ` (Цена)
- `КОМ-СИЯ` (Комиссия агентства)
- `АГЕНТ` (Имя агента должно совпадать с файлом сотрудников)
- `СТАТУС` или `на задатке` (на регистрации / на оплате / ждём выставления / расторгнут / дата поступления денег)
- `ЗАДАТОК` (Дата)
- `ЭД` или `Тип` (Тип договора: Застройщик, Вторичка и т.д.)

## 3. cash_flow.csv (Финансы)
**Источник:** Вкладка `ВЛОЖЕНИЯ AB 2025`
Обязательные колонки:
- `Тип` (Приход/Расход)
- `Сумма`
- `Статья`
- `Дата`
- `Счет`

## 4. accounts.csv (Счета)
Можно создать вручную.
Колонки: `name`, `type`, `balance`.

## Как запустить импорт
Когда файлы будут в папке, я сам запущу скрипт, или вы можете выполнить:
```bash
npx tsx scripts/import_data.ts
```
